Проект 'Допомога центру "Джерело"'

Автор:

Андрій Козак (Інші оголошення користувача)

Необхідний набір одягу для волонтерського центру джерело, найбільша потреба в теплих куртках, шкарпетках, зимових штанах, також необхідна їжа і мінеральна вода.

Редагувати Додати нову потребу

Перелік потреб:

Необхідні зимові штани для центру "Джерело".
